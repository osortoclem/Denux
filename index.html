Demostración de Favicon (Fabicom)

Un favicon es una pequeña imagen que aparece en la pestaña del navegador, marcadores y otras interfaces.

Códigos listos para usar:

Favicon PNG (32x32): usado por la mayoría de navegadores modernos.

<link rel="icon" type="image/png" sizes="32x32" href="https://ejemplo.com/favicon-32x32.png">

Favicon PNG (16x16): tamaño clásico, útil en marcadores y pestañas pequeñas.

<link rel="icon" type="image/png" sizes="16x16" href="https://ejemplo.com/favicon-16x16.png">

Favicon SVG: imagen vectorial escalable, ideal para pantallas de alta resolución.

<link rel="icon" type="image/svg+xml" href="https://ejemplo.com/favicon.svg">

Favicon ICO: compatible con navegadores antiguos (como Internet Explorer).

<link rel="shortcut icon" href="https://ejemplo.com/favicon.ico">

Apple Touch Icon: para pantallas de inicio en dispositivos iOS.

<link rel="apple-touch-icon" sizes="180x180" href="https://ejemplo.com/apple-touch-icon.png">

Manifest JSON: archivo para Progressive Web Apps (PWA).

<link rel="manifest" href="https://ejemplo.com/site.webmanifest">

Windows Tile: ícono y color para la pantalla de inicio en Windows.

<meta name="msapplication-TileColor" content="#ffffff">
<meta name="msapplication-TileImage" content="https://ejemplo.com/mstile-144x144.png">

Color de tema: define el color de la interfaz del navegador en móviles.

<meta name="theme-color" content="#ffffff">
